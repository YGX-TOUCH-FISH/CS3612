# 小作业环境配置与作业提交

## 环境配置

本次机器学习的小作业均使用.ipynb格式发放，请直接在.ipynb文件中含有TODO的代码块中填写相应代码并自测运行。此外，有Answer栏的Markdown块均为需要填写的简答题，请勿遗漏。小作业所需训练算力均较少，以辅助同学们熟悉流程、了解算法为主。

作业中使用的环境主要如下：

* Python 3.6

* scipy

* numpy

* scikit-learn

* matplotlib

* pandas

  助教测评时使用的完整环境列表请参照environment.yaml。

  下发的.ipynb文件中默认./dataset/AssignmentN/ 为对应作业的训练数据文件夹。

  本地测试中，推荐使用miniconda的虚拟环境进行python的环境版本管理，初始化指令如下：

  ```cmd
  conda create --name YOUR_NAME_OF_ENV python=3.6
  activate YOUR_NAME_OF_ENV
  conda install pandas matplotlib scikit-learn jupyter notebook
  pip install --upgrade scipy 
  ```

  如果下载时间过长，可以使用`conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free  `指令加载清华源或其他镜像源作为下载路径。

  初始化配置完成后，你可以在YOUR_NAME_OF_ENV环境下（`activate YOUR_NAME_OF_ENV`以进入环境）使用`jupyter notebook`指令开始对.ipynb文件的运行与编辑。

## 作业提交

作业完成后，请将你的.ipynb文件通过canvas系统提交。
